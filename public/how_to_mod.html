How to mod Ballistic Chickens PC

Mods Github

Steps

1. Go and find a trustable mod source(you can use our github)
2. Find were it says "Mod Code" in the readme.md file or if it tells you to go to another file and copy the code from there
3. Copy it to your clipboard
4. Load your copy of Ballistic Chickens PC Edition
5. Press Enable Modding
6. Press Launch
7. Paste THe Code You Copied Into The Popup
8. Press OK
9. Have Fun
